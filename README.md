# Automating API Management for Flex Gateway
This repository contains documentation and resources for automating API Management for Anypoint Flex Gateway with the Anypoint Platform Command Line Interface (CLI). The documentation guides you through key CLI commands for cataloging APIs on Anypoint Exchange, registering them in API Manager, deploying them to Flex Gateway, and applying policies. The concepts, capabilities, and examples covered in this repository lay the groundwork for building CI/CD pipelines.
